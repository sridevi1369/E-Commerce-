# Modern Shop - Vanilla JS E-commerce Application

This is a fully responsive, single-page E-commerce application built using **HTML, CSS, and Vanilla JavaScript**. No external frameworks or libraries were used (except for Ionicons for icons).

## Features

-   **SPA Experience**: Seamless navigation between Home, Products, Orders, and Tracking pages without reloading.
-   **Product Management**:
    -   Dynamic product listing with search and sort functionality.
    -   Flash Deals section with countdown timer.
    -   Product details modal with reviews, ratings, and "Add to Cart".
    -   Labels for "Bestseller", "Recommended", and "Flash Deal".
-   **Shopping Cart**:
    -   Add/Remove items, update quantities.
    -   Real-time total calculation.
    -   Persisted in `localStorage`.
-   **Order System**:
    -   Place orders and view order history.
    -   Track orders with a visual timeline (Ordered -> Packed -> Shipped -> Delivered).
    -   Persisted in `localStorage`.
-   **UI/UX**:
    -   Modern, clean design with soft colors and shadows.
    -   **Dark/Light Mode** toggle.
    -   Fully responsive (mobile-friendly).
    -   Toast notifications and loading animations.

## Setup

1.  Simply open `index.html` in any modern web browser.
2.  No build step or server required.

## Project Structure

-   `index.html`: Main HTML structure containing all views (Home, Products, Cart, Orders).
-   `style.css`: All styles, animations, and responsive rules.
-   `script.js`: Application logic, state management, and DOM manipulation.
